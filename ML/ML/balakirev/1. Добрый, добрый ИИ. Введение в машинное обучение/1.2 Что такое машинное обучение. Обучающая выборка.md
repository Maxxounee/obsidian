
* <mark>Апроксимация</mark> — приближение функции к данным. Научный метод, заключающийся в замене сложных математических объектов, функций или данных более простыми, близкими к оригиналу
* <mark>Образ</mark> — рассматриваемый объект в данных, совокупность его признаков
* <mark>Признак</mark> — отдельная характеристика объекта
* <mark>Вектор признаков</mark> — массив значений признаков рассматриваемого объекта

### Множества целевых значений и задачи, где они могут быть использованы:

| Множество                                | Задачи                                                                             | Пример                                                       |
| ---------------------------------------- | ---------------------------------------------------------------------------------- | ------------------------------------------------------------ |
| {-1, +1}                                 | Целевые значения для бинарной классификации                                        | Ебанет/не ебанет                                             |
| {1, 2, ..., M}                           | Целевые значения для M-классовой непересекающейся классификации                    | Различение лиц людей по рассам                               |
| R (вещественные числа)                   | Целевые значения для задачи регрессии. Предсказание конкретного числового значения | Прогноз курса валюты                                         |
| R^m (m-мерный вектор вещественных чисел) | Целевые значения для m-мерной задачи регрессии                                     | Прогноз местоположения в пространстве движущегося автомобиля |
