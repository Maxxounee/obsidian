<center>Признаковое пространство {f(x)} </center>

![[../../../assets/ml/balakirev/1/11.png|center]]
<center>Здесь явно нелинейная зависимость y(x). Поэтому, чтобы оставаться в рамках линейной модели при решении подобных задач, мы расширим признаковое пространство следующими преобразованиями:</center>
$$
\begin{align}
&f_{0}(x)=1 \\
&f_{1}(x)=x \\
&f_{2}(x)=x^{2} \\
&\dots \\
&f_{n}(x)=x^{n}
\end{align}
$$
<center>Линейно независимые признаки. То есть они НЕ выражаются один через другой линейно</center>

Теперь функция приобретает следующий вид (отличный от `a(x) = kx + b`):
$$
\begin{align}
&a(x)=1*\theta_{1}+x*\theta_{2}+x^{2}*\theta_{3}+\dots+x_{n}*\theta_{n+1} \\
&[\theta_{1},\dots,\theta_{n+1}]^{T}\text{ — Вектор параметров theta}
\end{align}

$$
<center>Вектор параметров theta (вектор коэффициентов) мы находим при обучении</center>

![[../../../assets/ml/balakirev/1/12.png|center]]<center>Если взять полином первой степени (n=1; a(x) = theta + theta * x), то получим аппроксимацию точек линейной функцией.Увеличивая порядок полинома, мы все точнее и точнее будем описывать экспериментальные зависимости.</center>

<blockquote>Линейная модель — это линейная зависимость <mark>признаков от параметров</mark> модели. Признаки — x, параметры — theta.</blockquote>

```python
import numpy as np

np.random.seed(0)
x = np.arange(-1.0, 1.0, 0.1) # аргумент [-1; 1] с шагом 0,1 

size_train = len(x) # размер выборки
w = [0.5, -0.3] # коэффициенты модели
model_a = lambda m_x, m_w: (m_w[1] * m_x + m_w[0]) # модель
loss = lambda ax, y: (ax - y) ** 2 # квадратическая функция потерь

a = model_a(x, w)
y = a + np.random.normal(0, 0.1, len(x)) # целевые значения

#_____ Код
Q = sum(loss(a, y)) / size_train
```

```python
import numpy as np

np.random.seed(0) 
x = np.arange(-1.0, 1.0, 0.1) # аргумент [-1; 1] с шагом 0,1

model_a = lambda xx, ww: (ww[0] + ww[1] * xx) # модель kx + b
Y = -5.2 + 0.7 * x + np.random.normal(0, 0.1, len(x)) # вектор целевых значений

# __ Код

# Матрица 20х1 из единиц
ones_column = np.ones(x.shape[0], dtype=int)
# Матрица 20х2 [[1, x], [1, x] ...]
X = np.column_stack([ones_column, x])
# Транспонированная матрица (повернутая)
X_T = X.transpose()
X_T_X = X_T @ X # Порядок важен. X @ X_T
# Обратная матрица
X_T_X_1 = np.linalg.inv(X_T_X)

# Дальше по формуле нахождения вектора w
X_T_y = X_T @ Y
w = X_T_X_1 @ X_T_y
```

```python
import numpy as np  
  
np.random.seed(0) # псевдослучайные числа образуют одну и ту же последовательность (при каждом запуске)  
x = np.arange(-1.0, 1.0, 0.1) # аргумент [-1; 1] с шагом 0,1  
  
  
model_a = lambda xx, ww: (ww[0] + ww[1] * xx + ww[2] * xx ** 2 + ww[3] * xx ** 3) # модель  
Y = np.sin(x * 5) + 2 * x + np.random.normal(0, 0.1, len(x)) # вектор целевых значений  
  
X = np.array([[1, xx, xx**2, xx**3] for xx in x]) # обучающая выборка для поиска коэффициентов w модели a  
  
# здесь продолжайте программу  
w = np.linalg.inv(X.T @ X) @ X.T @ Y
```
