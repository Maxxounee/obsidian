# Умножение

`*` — Поэлементное умножение
`@` — Матричное умножение `(np.dot(A, B, ...)` (dot product / matrix multiplication). Количество строк/столбцов первой матрицы должно совпадать с количеством столбцов/строк второй матрицы по правилу умножения матриц

```python
import numpy as np  
  
a = np.array([1, 2, 3])  
b = np.array([4, 5, 6])  
  
print(a * b)  # [4, 10, 18]
print(a @ b) # 1x4 + 2x5 + 3x6 = 32

# Квадратные матрицы
A = np.array([[1, 2], 
			  [3, 4]])  
B = np.array([[5, 6], 
			  [7, 8]])  

print(A @ B)  
# [[19 22]  - (1x5 + 2x7 = 19) |  (1x6 + 2x8 = 22)
#  [43 50]] - (3x5 + 28 = 43)  |  (3x6 + 4x8 = 50)


# Матрицы разных размеров.
A = np.array([[1, 2, 3],  
			  [4, 5, 6]])
  
B = np.array([[7, 8],  
			  [9, 10],  
			  [11, 12]])
  
print(A @ B)  
# [[ 58  64]
#  [139 154]]
print(B @ A)
# [[ 39  54  69]
#  [ 49  68  87]
#  [ 59  82 105]]
```
